## Todra

Todra is built around an **AI assistant** that understands what you want done and acts on it for you. You talk in plain language: it can **create, update, archive, and list** commitments and **habits** using built-in tools, then show you exactly what it asked for and what changed—so you stay in control without living inside a rigid checklist.

**Bring your own key (BYOK):** you pick the model provider, add your own API key, and it is stored only in **on-device secure storage**. Your tasks and habits still live locally; there is no Todra account and no company-hosted copy of your life list.

Optional reminders and JSON export/import are there when you want them, but the heart of the experience is **delegating the busywork to an assistant** instead of tapping through another generic to-do grid.
